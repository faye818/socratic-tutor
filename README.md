🎓 Socratic Tutor AI (苏格拉底启发式教育机器人)
本项目是一款基于 Gemini 2.0 Flash-Lite 模型构建的教育类智能体（Agent）。

🌟 产品核心逻辑
苏格拉底式教学法：通过引导性提问而非直接告知答案，激发学生的自主思考能力。
ZPD 理论应用：对话逻辑旨在识别学生的“最近发展区”，提供恰到好处的脚手架支持。

🛠️ 技术栈
Backend: Python 3.13
LLM: Google Gemini API (gemini-flash-lite)
Framework: Streamlit
Security: 采用环境密钥分离架构，通过 GitHub Secrets 安全部署。
